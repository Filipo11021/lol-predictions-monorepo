# Discord lol predictions
## What's inside?

This monorepo with Turborepo for discord lol predictions apps includes the following packages/apps:

### Apps and Packages

- [frontend](./apps//frontend/): a Next.js app for predictions data visualization
- [discord-bot](./apps/discord-bot/): discord bot for lol predictions
- `@repo/database`: prisma schema and prisma commands
- `@repo/typescript-config`: `tsconfig.json`s used throughout the monorepo

### Build

To build all apps and packages, run the following command:

```
cd my-turborepo
pnpm build
```

### Develop

To develop all apps and packages, run the following command:

```
cd my-turborepo
pnpm dev
```
